# ScrapeTrack-Licenta

 Aplicație de Web Scraping pentru monitorizarea prețurilor produselor de pe:

- eMAG
- Altex
- Flanco

 Tehnologii
 
- Python
- BeautifulSoup
- Requests
- Flask
- MySQL
- React.js

 Funcționalitate

- Căutare produse
- Salvare în MySQL
- Istoric prețuri
- Afișare date în tabele și grafice
- Scraping Automat
